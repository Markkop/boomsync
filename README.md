# BoomSync

Synchronized timer and player shuffle tool for coordinating activities between two rooms.

## Features

- Multiple timers with configurable rounds (3-5)
- Player shuffle to randomly assign teams to Room A and Room B
- Real-time P2P synchronization between devices
- Sound alerts and fullscreen timer mode

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
